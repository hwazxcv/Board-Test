# 프로젝트 소개
## 🪄 프로젝트 주제

Spring Boot를 활용하여 개별 회원이 이용할 수 있는 개인 게시판 프로젝트

## 🪄 주요기능
- Spring Security와 JWT를 이용한 회원가입 로그인
- 게시판 관련 파일 및 이미지 등록
- 댓글 및 개인 메일 보내기

## 💻 개발 환경
- IntelliJ 
- JAVA 21
- Spring Boot 3.x.x
- Oracle
- Gradle
